This implementation is based upon the MNIST letter recognition framework, in particular the following sources where used:
# http://neuralnetworksanddeeplearning.com/
# https://www.youtube.com/watch?v=8bNIkfRJZpo
# https://www.youtube.com/watch?v=d9hLNUzLBYI

To run this project follow the order:

1 - Generate training and test data via "generate_data.py" .
2 - Train the neural network via "tain_neural_network.py". 
3 - Test the trained network via "solve_ode.py" .

Tests of all files can be run via the ".py"-files in the TESTS folder.
